Project Overview

Restaurant Recommendation System Based on Fuzzy Sugeno Logic

This project implements a restaurant recommendation system using Fuzzy Sugeno Logic. The system evaluates restaurants based on multiple criteria and provides personalized recommendations to users.

Fuzzy logic is particularly useful for this application as it can handle imprecise information and linguistic variables, making it ideal for modeling human preferences and decision-making processes.

Fuzzy Sugeno Method

Understanding the inference process

The Sugeno fuzzy inference method, introduced by Michio Sugeno, is similar to the Mamdani method in many respects. The first two parts of the fuzzy inference process, fuzzifying the inputs and applying the fuzzy operator, are exactly the same.

The main difference is that the Sugeno output membership functions are either linear or constant. This makes the Sugeno method computationally efficient and works well with optimization and adaptive techniques.

In our system, we use a weighted average for defuzzification to calculate the final recommendation score.
